Row ID => 每筆的唯一ID
Order ID => 每個客戶的訂單D
Order Date => 產品的訂購日期
Ship Date => 產品的發貨日期
Ship Mode => 客戶指定的運送模式
Customer ID => 用於識別每個客戶的唯一ID
Customer Name => 客戶名稱
Segment => 客戶所屬的分類
Country => 客戶的居住國
City => 客戶的居住城市
State => 客戶的居住州
Postal Code => 每個客戶的郵政編碼
Region => 客戶所屬的地區
Product ID => 產品的唯一ID
Category => 訂購產品的類別
Sub-Category => 訂購產品的子類別
Product Name => 產品
Sales => 產品售價
Quantity => 產品數量
Discount => 提供折扣
Profit => 產生的利潤/損失
