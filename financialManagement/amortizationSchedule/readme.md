財務管理的公式好多都涉及大量求和，比方說算年金現值會用到的年金現值利率因子：
```math
PVIFA(k,n)=\Sigma_{1}^{n}\frac{1}{(1+k\%)^n}
```

雖然經過一連串的化簡可以整理成這樣：
```math
PVIFA(k,n)=\frac{(1+k\%)^n-1}{k\%(1+k\%)^n}
```

不過有時候總感覺化簡後的公式比起原式要難背，反正考試會用工程計算機？到時候注意要弄到能夠直接算 $\Sigma$ 的型號應該就可以了
